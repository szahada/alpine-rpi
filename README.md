# alpine-rpi

headless deploy in memory alpine linux on raspberry pi

## Getting Started

### GPIO

![alt pi0-pinout](https://github.com/szahada/alpine-rpi/blob/main/pi0-pinout.png)

### Prerequisites

* raspbery pi with sd card
* linux machine with internet access
* root

### Installing

* clone repo
* use sample.cfg file or create your own

### Executing program

* put sd card to your linux machine
* run installer
```
sudo ./install.sh -f sample.cfg
```
* read messages and confirm
* for more options run
```
./install.sh -h
```
enjoy
